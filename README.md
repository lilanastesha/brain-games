### Description
Set of 5 mathematical games. You must answer correctly three times in a row to win each game. If the answer is wrong, you can start any game again.
#### Games:
+ brain-even
+ brain-calc
+ brain-gcd
+ brain-progression
+ brain-prime

**You just need to write the game name in the terminal**
